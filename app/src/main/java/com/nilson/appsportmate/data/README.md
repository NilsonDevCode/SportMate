En esta carpeta se guardarán las reglas que determinan como la aplicación creará, guardará y modificará los datos.

Esta capa está hecha de repositorios que contienen los `data sources`, que son las fuentes de datos. Se debe crear un repositorio por cada fuente de datos que se tenga en la aplicación.

Por ejemplo, se podría crear un `MoviesRepository` para datos relacionados con las películas, o un `PaymentsRepository` para los datos relacionados con los pagos.
